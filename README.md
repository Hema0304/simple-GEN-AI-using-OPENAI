** Simple Generative AI using OpenAI **


This project is a basic implementation of Generative AI using OpenAI APIs, designed to demonstrate how AI models can generate human-like responses based on user input.

It serves as a beginner-friendly introduction to building AI-powered applications such as chatbots, assistants, and content generators.

Generative AI allows machines to create content, answer questions, and simulate conversations using Large Language Models (LLMs). These models process user input and generate meaningful responses in real time.

This project focuses on building a simple interaction system that connects user input with OpenAI’s API to produce intelligent outputs. Similar minimal applications often use lightweight setups with Python and environment variables to securely access the API.

## Features

- AI-powered text generation
- 
- Interactive user input and response
- 
- Fast and simple implementation
- 
- Natural language understanding
- 
- Secure API key handling using environment variables
- 
- Minimal and beginner-friendly codebase

## Tech Stack
Python
OpenAI API
dotenv (for environment variables)
Jupyter Notebook / Python Script


## How It Works
-User enters a prompt or query

-The input is sent to the OpenAI API

-The model processes the request

-A generated response is returned

-Output is displayed to the user
